# Problem

The UI for activating or suspending an organization does not work correctly!

# Context

When activating or suspending an organization a dialog appears where the user can click
to suspend or activate. After clicking that button it appers nothing is happening.

In reality the system does suspend or activate but the dialog just stays there.

# Except

1. The action buttons should be disabled while the backend is working
2. The dialog should close after the backend returns with success.
3. The organization data should be refreshed.
